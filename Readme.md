## 📱 Mobile App (Android)

Vault Opus Android is the native mobile implementation of the Infinity Cloud Storage Project. It leverages Capacitor and Chaquopy to run the powerful Vault Opus backend directly on your smartphone, giving you secure, unlimited storage on the go.

### 📥 Installation
1. Download the latest `app-debug.apk` from the releases or build artifacts.
2. Open the APK on your Android device.
3. If prompted, allow installation from "Unknown Sources."
4. Launch **VAULT OPUS** from your app drawer.

> [!TIP]
> **Desktop User?** If you are looking for the Linux, Windows, or macOS version, please visit our dedicated desktop repository: **[VAULT-OPUS-Desktop](https://github.com/WeDu-official/VAULT-OPUS-Desktop)**.

## ✨ Features

- **Unlimited Storage**: Leverages Discord's infrastructure for data storage.
- **Advanced Encryption**: Secure your files with Zero-Knowledge encryption.
- **Native Experience**: Optimized UI/UX specifically for mobile touchscreens.
- **Integrated Python Engine**: Runs the full core logic via Chaquopy for maximum security and performance.
- **Atomic Operations**: Support for Soft and Hard atomic uploads for data integrity.

---

## 📋 System Requirements

- **Android Version**: `7.0` (API 24) or higher.
- **Architecture**: Supports `armeabi-v7a`, `arm64-v8a`, `x86`, and `x86_64`.
- **Internet**: Required for Discord API communication.
- **Permissions**: Requires Storage access (for file management) and Internet access.

---

## 🛠 Technical Stack

- **Core**: Python 3.11 (via Chaquopy)
- **Frontend**: React 18+ & Tailwind CSS
- **Bridge**: Capacitor JS
- **Database**: PowerDB (High-performance metadata storage)
